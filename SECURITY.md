# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please email:

**security@x1nexus.com** (placeholder - update with real contact)

Or create a security advisory on GitHub:
https://github.com/omenpotter/vesting-halving-program/security/advisories/new

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response Time

We aim to respond within 48 hours and provide updates every 72 hours until resolved.

## Security Features

- **Time-Locked Vesting**: Cannot be bypassed or rushed
- **Immutable Supply**: Total supply calculated upfront
- **Authority Controls**: Only beneficiary can claim
- **Overflow Protection**: Safe arithmetic operations
